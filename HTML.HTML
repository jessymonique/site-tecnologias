O que é HTML?

HTML (HyperText Markup Language) é a linguagem padrão usada para criar páginas na web. Ele serve para estruturar o conteúdo de um site, definindo elementos como títulos, parágrafos, imagens, links, tabelas e listas. O HTML utiliza tags (ou marcadores), que indicam ao navegador como exibir cada parte do conteúdo. Por si só, o HTML não tem funcionalidades visuais avançadas — para isso, é normalmente combinado com CSS (para estilos) e JavaScript (para interatividade). Simples e essencial, o HTML é o primeiro passo para quem deseja aprender desenvolvimento web.
